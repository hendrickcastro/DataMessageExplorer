# DataMessage Explorer

Cross-platform desktop application for managing Azure Cosmos DB, Service Bus, and Redis.

## Downloads

Check the [Releases](https://github.com/hendrickcastro/DataMessageExplorer/releases) page for the latest version.

### Platforms
- **Windows**: Download `DataMessageExplorer-windows.zip`
- **macOS**: Download `DataMessageExplorer-macos.zip`

## Installation

1. Download the zip file for your platform from [Releases](https://github.com/hendrickcastro/DataMessageExplorer/releases)
2. Extract the archive
3. Run the application:
   - **Windows**: Double-click `DataMessage Explorer.exe`
   - **macOS**: Open `DataMessage Explorer.app`

## Auto-Update

The application automatically checks for updates on startup. When a new version is available, you'll be prompted to download and install it. You can also manually check via the **Check Updates** button in the toolbar.

## Features

- **Azure Cosmos DB**: Browse databases, containers, query documents, insert/update/delete
- **Azure Service Bus**: Browse queues/topics, peek messages, send messages, manage scheduled messages
- **Redis**: Browse keys, inspect values
